This project implements automated API tests for
the [QA Sandbox — Social Network for Test Automation Practice](https://github.com/manikosto/qa-automation-sandbox). The
tests are written using **Python**, **Pytest**, **Allure**, **Pydantic**, **Faker** and **HTTPX**. The test
application’s source code is available on [GitHub](https://github.com/manikosto/qa-automation-sandbox).
